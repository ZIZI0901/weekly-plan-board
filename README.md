# Weekly Plan Board

一个纯前端的周计划/日计划看板，支持周待办、日待办、完成项、月视图、颜色标记和云端同步。

已部署网站：

https://illustrious-liger-eec5eb.netlify.app/

## 功能

- 周待办和日待办管理
- 已完成事项归档
- 月视图和日期跳转
- 日期颜色标记
- 本地浏览器存储
- Supabase 登录与云端同步

## 使用

直接用浏览器打开 `index.html` 即可。

也可以访问线上版本：

https://illustrious-liger-eec5eb.netlify.app/
